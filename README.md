# 🌱 Monitoramento de Umidade do Solo com Arduino

Projeto desenvolvido utilizando Arduino e simulado totalmente online através do  
[Tinkercad](https://www.tinkercad.com/?utm_source=chatgpt.com).

O sistema realiza a leitura da umidade do solo utilizando um sensor de umidade, exibindo os valores em um display LCD 16x2. Também possui um botão para ativar/desativar o monitoramento e um LED indicador de funcionamento.

---

# 🚀 Funcionalidades

- ✅ Leitura da umidade do solo
- ✅ Conversão da leitura analógica para porcentagem
- ✅ Exibição da umidade em um display LCD
- ✅ Botão para ligar/desligar o sistema
- ✅ LED indicador de status
- ✅ Simulação 100% online no Tinkercad

---

# 🛠️ Tecnologias Utilizadas

- Arduino UNO
- Linguagem C/C++
- Sensor de Umidade do Solo
- Display LCD 16x2
- Tinkercad
- Biblioteca `LiquidCrystal`
